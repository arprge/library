О библиотеке

О проекте

Данный сайт представляет собой учебный проект для первого курса университета. Целью является создание функционального веб-приложения для управления личной библиотекой книг.

Проект разработан с использованием базовых веб-технологий: HTML5, CSS3 и JavaScript, демонстрируя фундаментальные принципы веб-разработки.

Функциональность

  • Каталог книг: Просмотр доступной коллекции книг
  • Поиск и фильтрация: Быстрый поиск по названию, автору или жанру
  • Списки чтения: Управление прочитанными книгами и списком желаемого
  • Статистика: Отслеживание количества прочитанных книг
  • Адаптивный дизайн: Корректное отображение на различных устройствах

Технологии

HTML5

Структура и семантика страниц

CSS3

Стилизация и адаптивный дизайн

JavaScript

Интерактивность и работа с данными

LocalStorage

Сохранение пользовательских данных

Как пользоваться

  1. Перейдите на страницу Поиск книг для просмотра каталога
  2. Используйте поиск и фильтры для нахождения интересующих книг
  3. Добавляйте книги в списки "Прочитано" или "Хочу прочитать"
  4. Просматривайте свои списки на странице Мои книги
  5. Отслеживайте статистику прочтения
